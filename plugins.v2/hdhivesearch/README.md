# HDHive资源搜索插件

通过HDHive Open API搜索网盘资源，支持115/123/夸克/百度网盘等多种网盘类型。

## 功能特性

### 核心功能
- 🔍 **智能搜索** - 通过影片名/TMDB ID搜索网盘资源，支持请求去重
- 📋 **详情展示** - 显示资源大小、分辨率、字幕、来源等详细信息
- 🔓 **积分解锁** - 使用积分或VIP免费额度解锁资源
- 👤 **用户管理** - 查询积分、VIP状态、每日签到
- ✅ **每日签到** - 签到获取积分奖励
- 📊 **免费额度** - VIP用户查看每周免费解锁额度

### v2.0.0 新增功能
- 🚀 **CMS自动转存** - 支持自动转存到115网盘/阿里云盘/夸克网盘/百度网盘
- 📈 **统计系统** - 完整的搜索/转存统计，成功率分析
- 🔄 **智能重试** - 失败自动重试机制，提高成功率
- 📱 **微信优化** - 专门优化的微信消息格式
- ⚡ **优先级配置** - 自定义网盘类型优先级排序
- 👑 **Premium控制** - VIP用户专属功能和开关
- 🎯 **健康检查** - 详细的诊断报告和修复建议

## 安装方法

1. 在MoviePilot设置中找到「插件」→「插件源」
2. 添加插件源地址
3. 在插件市场搜索「HDHive资源搜索」
4. 点击安装并配置API Key

## 配置说明

| 配置项 | 说明 |
|--------|------|
| API Key | HDHive Open API密钥（必填） |
| API地址 | HDHive API服务地址（默认：https://hdhive.com/api/open） |
| 发送通知 | 是否发送搜索结果通知 |
| Premium用户 | 是否为Premium用户（开启后可使用VIP专属功能） |
| 网盘优先级1-4 | 自定义网盘类型显示优先级（115/quark/123/baidu等） |
| 启用CMS转存 | 是否启用CMS自动转存功能 |
| CMS地址 | CMS服务地址（如：http://localhost:8085） |
| CMS用户名 | CMS登录用户名 |
| CMS密码 | CMS登录密码 |

## 使用方法

### 基本搜索
发送「影片名？」进行搜索，例如：
```
武林外传？
权力的游戏？
复仇者联盟？
```

### 查看详情
搜索结果后，回复数字查看详情：
```
1？      # 查看第1个资源详情
2？      # 查看第2个资源详情
```

### 指定网盘类型
```
1.115？   # 查看115网盘资源
1.quark？ # 查看夸克网盘资源
```

### 解锁资源
查看详情后，回复「解锁」使用积分解锁

### 用户命令
| 命令 | 说明 |
|------|------|
| /hdhive_me | 查看用户信息（积分、VIP状态等） |
| /hdhive_checkin | 每日签到获取积分 |
| /hdhive_quota | 查看VIP免费额度 |
| /hdhive_search | 搜索资源 |
| /hdhive_stats | 查看统计信息（搜索/转存数据） |

## API接口说明

插件暴露以下API接口供外部调用：

### 搜索资源
```
GET /plugin/HDHiveSearch/search?tmdb_id=550&media_type=movie
```

### 解锁资源
```
POST /plugin/HDHiveSearch/unlock
Content-Type: application/json
{"slug": "resource-slug"}
```

### 用户信息
```
GET /plugin/HDHiveSearch/user
```

## 注意事项

1. 需要先在HDHive官网注册并获取API Key
2. 部分功能需要VIP会员才能使用
3. API有调用频率限制，请合理使用
4. 与ChatGPT插件可能存在冲突，建议调整插件优先级

## CMS自动转存配置

插件支持在解锁资源后自动转存到网盘，需要配置CMS（如Alist等）：

### 支持的网盘类型
- 115网盘
- 阿里云盘
- 夸克网盘
- 百度网盘

### 配置步骤
1. 在插件配置中启用"启用CMS转存"
2. 填写CMS地址、用户名和密码
3. 保存配置后，解锁资源时会自动转存

### 转存规则
- 资源链接包含"115"时转存到115网盘
- 资源链接包含"aliyundrive"时转存到阿里云盘
- 资源链接包含"quark"时转存到夸克网盘
- 资源链接包含"baidu"时转存到百度网盘
- 其他类型不转存

## 统计系统

插件自动记录以下统计数据：
- 总搜索次数
- 成功/失败搜索次数
- CMS转存次数
- 成功/失败转存次数
- 转存成功率
- 最后活动时间

使用 `/hdhive_stats` 命令查看统计信息。

## 网盘优先级配置

可以自定义网盘类型的显示优先级，搜索结果会按优先级排序：
- 优先级1：最高优先级
- 优先级4：最低优先级

支持的网盘类型：`115`、`quark`、`123`、`baidu`、`aliyundrive`

## Premium功能

Premium用户可享受以下特权：
- 使用VIP专属接口
- 更高的调用频率限制
- 优先获得新功能

在插件配置中开启"Premium用户"开关即可启用。
