## 🎉 版本概述

HDHive插件 v2.0.0 是一个重大功能更新版本，引入了多项核心功能，大幅提升了插件的自动化程度和用户体验。本版本重点增强了CMS集成、统计分析、智能重试等企业级功能。

## ✨ 核心新功能

### 1. 🚀 CMS自动转存功能

**功能描述**
在解锁资源后，插件会自动将资源转存到配置的CMS系统中，支持主流网盘类型。

**支持网盘**
- 115网盘
- 阿里云盘
- 夸克网盘
- 百度网盘

**配置方式**
```
1. 在插件配置中启用"启用CMS转存"
2. 填写CMS地址（如：http://localhost:8085）
3. 填写CMS用户名和密码
4. 保存配置即可自动生效
```

**智能识别**
插件会根据资源链接自动识别网盘类型：
- 链接包含"115" → 转存到115网盘
- 链接包含"aliyundrive" → 转存到阿里云盘
- 链接包含"quark" → 转存到夸克网盘
- 链接包含"baidu" → 转存到百度网盘

**错误处理**
- 转存失败自动重试（最多3次）
- 详细的错误提示和诊断信息
- 不影响正常的解锁流程

### 2. 📈 统计系统

**功能描述**
完整的统计分析系统，追踪插件使用情况，帮助用户了解使用效果。

**统计指标**
- 总搜索次数
- 成功搜索次数
- 失败搜索次数
- CMS转存次数
- 成功转存次数
- 失败转存次数
- 转存成功率（自动计算）
- 最后活动时间

**查看方式**
```
命令: /hdhive_stats
```

**数据持久化**
- 统计数据自动保存到配置
- 插件重启后数据不丢失
- 支持长期数据积累

### 3. 🔄 智能重试机制

**功能描述**
对失败的请求自动重试，提高操作成功率。

**重试场景**
- API调用失败
- CMS转存失败
- 网络超时
- 临时性错误

**重试策略**
- 最多重试3次
- 指数退避策略（1s, 2s, 4s）
- 仅对可重试错误进行重试
- 详细的重试日志记录

### 4. 📱 微信消息格式化优化

**功能描述**
专门为微信通知优化的消息格式，提升阅读体验。

**优化内容**
- 搜索结果格式化（使用emoji和分段）
- 详情信息格式化（突出关键信息）
- 统计信息格式化（数据可视化）
- 错误信息格式化（清晰的错误提示）

**效果**
- 消息更易读
- 关键信息突出
- 视觉层次清晰

### 5. ⚡ 网盘优先级配置

**功能描述**
自定义网盘类型的显示优先级，搜索结果按优先级排序。

**配置选项**
- 优先级1（最高）
- 优先级2
- 优先级3
- 优先级4（最低）

**支持的网盘类型**
- `115` - 115网盘
- `quark` - 夸克网盘
- `123` - 123网盘
- `baidu` - 百度网盘
- `aliyundrive` - 阿里云盘

**使用场景**
- 优先显示常用网盘类型
- 根据网盘速度调整优先级
- 根据存储空间调整优先级

### 6. 👑 Premium用户控制

**功能描述**
VIP用户专属功能和开关，提供更好的使用体验。

**Premium特权**
- 使用VIP专属接口
- 更高的调用频率限制
- 优先获得新功能
- 更好的服务质量

**配置方式**
```
在插件配置中开启"Premium用户"开关
```

### 7. 🎯 健康检查增强

**功能描述**
详细的诊断报告和修复建议，帮助用户快速解决问题。

**检查项目**
- API连接状态
- API密钥有效性
- CMS连接状态
- 网络状态
- 配置完整性

**诊断报告**
- 问题定位
- 原因分析
- 修复建议
- 参考链接
